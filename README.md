# ◈ Flutter & Mobile Development Quiz App

A modern, interactive Single Page Application (SPA) designed to test and review knowledge of Flutter and Mobile App Development.

## 🚀 Key Features
* **Comprehensive Question Bank:** Covers 10 chapters including Dart, Flutter Architecture, UI Widgets, and State Management.
* **Customizable Quizzes:** Select specific chapters or take a full mock exam.
* **Smart Review:** Immediate feedback on answers and a detailed performance summary at the end.
* **Adaptive Learning:** Option to retry only the questions you missed.
* **Browse Mode:** Explore all questions and answers without taking a quiz.
* **Print-Ready:** Specialized CSS for clean, professional hard-copy printing.

## 📚 Topics Covered
1.  **Mobile Basics:** Native vs. Hybrid vs. Web apps.
2.  **Architecture:** Layers, Engine (Skia), and Framework.
3.  **Widgets:** Material & Cupertino, Scaffold, and Layouts.
4.  **UI Components:** Text, Buttons, Images, and Inputs.
5.  **UX/UI Design:** Usability, Affordance, and Haptics.
6.  **Setup:** Flutter SDK, Android Studio, and Path configuration.
7.  **Dart Language:** Functions, OOP, and Data types.
8.  **Navigation:** Routes, Navigator (Push/Pop), and Data passing.
9.  **State Management:** Stateless vs. Stateful and Lifecycle methods.

## 🛠️ Tech Stack
* **HTML5 & CSS3:** Responsive Grid/Flexbox layout with Dark Mode.
* **Vanilla JavaScript:** Lightweight logic with no external dependencies.
